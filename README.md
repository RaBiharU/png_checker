# png_checker
任意の画像ファイル(1つ以上)を選んで、
- ファイル形式(PNGか)
- 幅(ピクセル)
- 高さ(ピクセル)
- DPI
- ファイルサイズ
が、それぞれ設定した規定通りかを確認できるアプリケーションです。

セトリイラストのpx数チェックや印刷用データのサイズ確認などに便利だと思います。

## 🖥️アプリを使いたい人向け(Python環境不要)

`png_checker.exe` (アプリ本体) があればPython環境が無くても実行できます。

### ダウンロード

[ここ(Releasesページ)](https://github.com/RaBiharU/png_checker/releases/tag/v1.0.0)から最新版の png_checker.exe をダウンロードしてください。

### 注意点

- 動作確認はWindows 11(64bit)でのみ行っています。
- Windowsで初回起動時、「WindowsによってPCが保護されました」という表示が出るかもしれませんが、[詳細情報] → [実行] で起動できます。(変なものは仕込んでないので大丈夫です！)

## ⚙️コードを改変・再ビルドしたい人向け(Python環境必要)

このリポジトリにあるソースコードを元に、自由に改変して使ってください。

### 必要なもの

- Python(3.12.4で開発)
- requirements.txt に記載のライブラリ
- png_checker.py(ソースコード本体)
- icon.ico(アイコンファイル)(一応、歌奏絆さんの髪の毛の色のカラーパレットです)

### 環境構築と再ビルドの手順

1. **リポジトリをクローンまたはダウンロード**

2. **Pythonライブラリのインストール**

    ターミナル（コマンドプロンプト）で、このフォルダに移動して以下を実行してください。
    ```
    pip install -r requirements.txt
    ```

3. **.exeのビルド**

    pyinstallerがインストールされたら、以下のコマンドでビルドできます。
    (dist フォルダの中に新しい png_checker.exe が生成されます)

    ```
    pyinstaller -F -w png_checker.py --icon=icon.ico
    ```

    なお、コードはChatGPTが作ってくれたものを手直ししただけです…
